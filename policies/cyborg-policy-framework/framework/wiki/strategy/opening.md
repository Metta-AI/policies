# Opening Strategy

Universal principles for the first 10-20% of any game.

## The Opening Objective

Every game has a critical opening window where early decisions compound. The universal
goal: **maximize information and option value before committing to a strategy.**

## Phase 1: Orientation (ticks 0-50)

What the agent must learn immediately:
1. **Where am I?** — map position, surroundings, starting resources
2. **What's nearby?** — resources, threats, objectives, other players
3. **What are the rules?** — if the game has variable mechanics per match, detect them
4. **What's the win condition priority?** — scoring, survival, elimination, objectives

## Phase 2: Opening Commitment (ticks 50-200)

After initial orientation, commit to an opening strategy:

| Game Type | Common Openers |
|---|---|
| Resource/economy | Rush a critical resource, secure income sources |
| Territory/area | Expand to claim key positions early |
| Military/combat | Scout opponent, choose aggression vs defense |
| Social/deduction | Establish information network, probe other players |
| Survival | Secure basic needs (food, shelter, tools) |

## Universal Opening Rules

1. **Scout before committing.** Don't mine the first resource you see — look around first.
   The second-closest resource might be twice as valuable.

2. **Exploit freebies.** Most games give starting resources or advantages. Use them
   immediately — they have zero opportunity cost.

3. **Don't over-explore.** Exploration has diminishing returns. After discovering key
   locations, start executing. The LLM should push toward execution by tick 100-200.

4. **Establish economy early.** Whatever the game's "income" is — resource gathering,
   territory control, card draw — set it up before everything else.

5. **Don't fight early.** Combat in the opening wastes resources and time. Avoid
   confrontation unless defending is forced.

## Opening Mistakes (universal)

- **Wandering**: exploring aimlessly past the orientation phase
- **Hoarding**: collecting resources without using them
- **Over-investing**: dumping all resources into one thing before understanding the game state
- **Ignoring the clock**: not recognizing when the opening phase is ending
- **Copying**: blindly repeating last game's opener without adapting to the current map/seed
